## Формат JSON-спека (вход)

```json
{
  "canvas": "front_cover",
  "image": "<путь к арту>",
  "image_px": [W, H],
  "elements": {
    "<role>": {
      "text": "строка текста",
      "box_pct": [x, y, w, h],      // в ПРОЦЕНТАХ от image_px, начало — левый-верхний угол
      "box_px":  [x, y, w, h],      // опционально дубль в пикселях; если есть — приоритет
      "align": "left|center|right",
      "valign": "top|middle|bottom",
      "color": "#RRGGBB",
      "scrim": false,                // подложка под текстом для читаемости
      "max_lines": 5,
      "font": null                   // путь к TTF или null → дефолт
    }
  }
}
```
Ролей может быть несколько (title, subtitle, author). В тест-кейсе — только `title`.

## Что написать

Файл: **`engine/text_layout.py`**

API:
```python
def render_layout(
    art_path: str,
    layout: dict | str,          # dict спека или путь к .json
    out_path: str,
    default_font: str | None = None,
) -> dict:                       # вернуть инфо по каждому элементу (кегль, строки, bbox)
    ...
```

Поведение:
1. Открыть арт (RGB). Прочитать спек (если строка — загрузить JSON).
2. Для каждого элемента посчитать рамку в пикселях: если есть `box_px` — взять его,
   иначе перевести `box_pct` в пиксели от `image_px`/реального размера арта.
3. **Авто-перенос строк** по словам под ширину рамки.
4. **Авто-подбор кегля**: начни с крупного и уменьшай, пока текст влезает в рамку по
   ширине И по высоте, и число строк ≤ `max_lines`. (Аналогично черновику архитектора.)
5. **Шрифт:** `font` из элемента → иначе `default_font` → иначе системный
   `DejaVuSans-Bold.ttf` (фолбэк, не падать).
6. **Выравнивание** по `align` (внутри рамки) и `valign` (блок текста по вертикали рамки).
7. **Цвет** из `color`. Если `scrim: true` — под текстом нарисуй полупрозрачную тёмную
   (или светлую — противоположную цвету текста) подложку с небольшим паддингом, чтобы
   текст читался на пёстром фоне. Если `scrim: false` — без подложки.
8. **Авто-контраст (страховка):** если `color` не задан, выбери чёрный/белый по средней
   яркости арта под рамкой (тёмный фон → белый текст, и наоборот). Если `color` задан —
   используй его.
9. Сохранить `out_path` (PNG). Вернуть словарь: на каждый элемент — итоговый font_size,
   число строк, фактический bbox текста в пикселях.
10. `if __name__ == "__main__":` — прогнать на тест-кейсе ниши:
    арт `data/niches/2026-06_test/sources/images/front_art_test.png`,
    спек `data/niches/2026-06_test/sources/layout_test.json`,
    выход `output/text_layout_test.png`. Вывести возвращённый словарь.

Зависимость: `Pillow` (есть). `output/` создавать программно, не коммитить.

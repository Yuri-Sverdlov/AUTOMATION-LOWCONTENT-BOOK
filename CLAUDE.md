# CLAUDE.md

Guidance for Claude Code (claude.ai/code) when working in this repo.

## Все рабочие правила — в `AGENTS.md`

Этот файл намеренно короткий. Источник правды по ролям, циклу работы, запретам и
техническим правилам — **`AGENTS.md`**. Прочитай его первым.

## Быстрый старт сессии

```
Прочитай CONTEXT.md, затем AGENTS.md, затем tasks/TASK.md — и выполни задание.
По завершении заполни tasks/REPORT.md.
```

## Карта проекта

- `PROJECT-OVERVIEW.md` — стратегия и идея конвейера low-content книг (KDP).
- `STRUCTURE.md` — формат папок данных (`data/`, `niche.yaml`, `published.csv`).
- `CONTEXT.md` — память проекта и текущий фокус.
- `tasks/TASK.md` ↔ `tasks/REPORT.md` — петля «задание ↔ отчёт».
- `PROJECT_LOG.md` — журнал сессий.
- `reference/` — эталоны от ранее выпущенных книг (для сверки).
- `reusable/` — отобранный код из прошлых проектов.
